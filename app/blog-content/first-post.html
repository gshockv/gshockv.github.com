Зачем понадобилось создать этот сайт

Можно начать с того, что мне просто захотелось это сделать. Но есть и основные цели:
  • показывать на главной большой баннер с проектом и картинкой
  • научиться делать сайты
  • сделать онлайн-портфолио своих проектов
  • вынести в онлайн резюме (CV)
  • сделать небольшой движок блога
  • освоить модные и молодежные технологии веба

Результат

Результат перед вами. Это статический сайт: имею ввиду, что он не привязан (по крайней мере пока) к бэк-енду. Вся логика построения страниц, навигация и прочее выполняется строго на клиент-сайде. Хостится сайт на GitHub Pages.

Из чего это все приготовлено:

  • GitHub Pages
  • AngularJS
  • underscore.js
  • Twitter Bootstrap
  • Node.js
Тут упомяну, что Node.js используется только для локального хостинга, т.е. при разработке и отладке скриптов и верстки

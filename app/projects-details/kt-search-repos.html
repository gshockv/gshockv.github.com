KtSearch Repos Android Application

KtSearchRepos - приложение, которое почти ничего не делает. Единственная полезная функциональность - поиск github-репозитория по введенному ключевому слову. Это приложение разработано во время ознакомления с замечательным языком программирования Kotlin и его применением в разработке Android-приложения.

Использованные технологии и инструментарий

  • Android SDK
  • Android Support Library
  • Android Material Design Design Support Library
  • Kotlin
  • Retrofit
  • GitHub Public API

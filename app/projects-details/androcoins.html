AndroCoins Android Application

Androcoins - простейшее приложение для ведения личных карманных расходов. Имеет гибко-настраиваемый список категорий расходов. С помощью разных типов временных отчетов можно посмотреть все свои расходы за выбранный период времени.

Использованные технологии и инструментарий

  • Android SDK
  • Java Programming language
  • SQLite
  • GitHub
